1.5. Processor SDK RTOSÂ¶

Processor SDK RTOS is a companion SDK to Processor SDK Linux on the J784S4 family of devices. In order to use some SoC features from the core running Linux, one needs to download and install the PSDK RTOS package.

Important

This package is NOT included in PSDK Linux installer and needs to be downloaded separately.
